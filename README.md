# Vimconf

## Brief 
 - vim configure
 - based on bundle

# Feature
 - c/c++ support
 - golang support
 - cscope support 
 - ctags support
 - python support
 - YoucompeteMe SUPPORT

## DEPEND
 - ack-grep


## Install

```   

 cp -r vim  ~/.vim
 cp vim/vimrc ~/.vimrc
 vim ~/.vimrc
 :BundleInstall

```
